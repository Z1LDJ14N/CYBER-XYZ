# XYZ BY ZILDJIAN

Website multifungsi sederhana, ringan, dan mobile-first.
Dibangun dengan HTML, CSS, dan Vanilla JavaScript.

## Fitur
- Text Tools (Counter, Case Converter)
- Generator (Password, Random)
- Developer Tools (JSON Formatter, URL Encode)
- Dark Mode Support
- 100% Client Side (Tanpa Backend)

## Cara Deploy ke Vercel (Gratis)

1. **Upload ke GitHub**
   - Buat akun GitHub.
   - Buat repository baru (misal: `xyz-web`).
   - Upload semua file (index.html, css folder, js folder, dll) ke repository tersebut.

2. **Connect ke Vercel**
   - Buka [Vercel.com](https://vercel.com).
   - Login dengan GitHub.
   - Klik "Add New" > "Project".
   - Pilih repository `xyz-web` yang baru dibuat.
   - Klik "Deploy".

3. **Selesai!**
   - Website kamu akan live di `https://xyz-web.vercel.app`.
